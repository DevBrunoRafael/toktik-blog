# toktik-blog

Projeto desenvolvido na disciplina de estrutura de dados 2, trata-se da criação de uma rede social utilizando estrutura de dados adequadas para construção desse tipo de sistema.

## Funcionalidades
- Login
- Cadastro de usuários
- Busca de publicações
- Criar publicações
- Busca de usuários
- Seguir usuários
- Seguidores e seguindo
- Filtrar publicações criadas
- Dark mode
